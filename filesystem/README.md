# Structuration des données produites à l'IRAMAT

Exemples de schémas/structures (_filesystem_ ou fs):

## fs1
> Utilisée par Terrifer

```
├───site1
│   ├───objet1
│   │   ├───metallo1
│   │   │       micrographie1.jpg
│   │   │       micrographie2.jpg
│   │   │       micrographie3.jpg
│   │   │       
│   │   ├───metallo2
│   │   │       micrographie1.jpg
│   │   │       micrographie2.jpg
│   │   │       
│   │   └───metallo3
│   │           micrographie1.jpg
│   │           
│   └───objet2
│       ├───metallo1
│       │       micrographie1.jpg
│       │       micrographie2.jpg
│       │       
│       └───metallo2
│               micrographie1.jpg
│               
└───site2
    ├───objet1
    │   ├───metallo1
    │   │       micrographie1.jpg
    │   │       micrographie2.jpg
    │   │       micrographie3.jpg
    │   │       
    │   ├───metallo2
    │   │       micrographie1.jpg
    │   │       micrographie2.jpg
    │   │       
    │   └───metallo3
    │           micrographie1.jpg
    │           
    └───objet2
        ├───metallo1
        │       micrographie1.jpg
        │       micrographie2.jpg
        │       
        └───metallo2
                micrographie1.jpg
```
